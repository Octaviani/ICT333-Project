Visualisation of Data

Data visualization is the presentation of data in a pictorial or graphical format; the modern equivalent of visual communication. By visualising the strength of relationships between artefacts, users would quickly view the information that will be the most relevant to their search


The concept of the data visualisation is derived of decomposed elements of Force-Directed Graph, it displays the occurence of keyterms that corresponds to the search query, mapping out the descended nodes. The size of each nodes corresponds to the frequency of word occurence.


Furthermore, this prototype aims to render this data visualisation into a 3D-form of interactive network of data, which enables the users to view and interact with the structure

Real-Time Search

The search engine enables the user to browse other user generated data synchronously. The results are presented along with the details of the metadata. The search features full-text search, hit highlighting, real-time indexing and dynamic clustering of data.


The search feature had been constructed through the use of open source enterprise search platform; Apache Solr, in conjunction with Lucene Library. This allows it to be highly scalable and fault tolerant


The search platform is designed to process and analyse rich, structured files, particularly documents that supports text-based file extensions such as PDF, DOCX, TXT and others.

Customised Prediction

This project aims to feature machine learning element into the prototype. The goal is to build an environment for scalable performance machine learning application by providing a customised recommendation of (potentially) relevant documents that fits individual users, based on their past activities


Potentially, this could be achieved through the integration of Apache Mahout and Apache Lucene/Solr. Which utilises the functionality called ItemRecommender and UserRecommender; using Solr for recommendation queries and Mahout to generate the similarity matrix model
