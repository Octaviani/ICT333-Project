  • Basics

  • How can I update my profile details?

    In the Profile panel click on Profile Details, this allows the user to update their basic details, including e-mail, firstname/lastname and their department/profession.

  • Can I change my password?

    No, the passwords had been generated for each users by the Administrator, hence users will not be able to change the passwords allocated to them.

  • How large can the file upload be?

    Currently, files to be uploaded cannot be larger than 100 megabytes per-upload.

  • What should I do if my file is larger than the size limit?

    If needed to upload large file, it is recommended to split the file into a smaller compressed file and uploading the segments individually.

  • How do I retrieve files

    Retrieving files can be done by simply clicking onto the content preview (or the BoxView) and click on File and Save As, this then prompts the user to save the file onto their local directory

  • Compatibility

  • What web-browsers are compatible with DataHive?

    To ensure you have the best experience as possible, it is recommended using the most up-to-date version of one of the following browsers: Mozilla Firefox, Safari, Google Chrome, Internet Explorer

  • I cannot see some of the web contents

    Try to check if all addons are enabled on your browser, which includes proprietary plugins such as Adobe Flash Player and Adobe Reader. Otherwise, if the issue prevails, check whether your browser supports HTML5, CSS3 and flash components; if not, update your browser to the latest version.

  • What kind of file is supported?

    The system supports most rich text files, this includes files that ends in the extention of .PDF, .TXT, .DOCX or .DOCS. However, it is still possible to upload non-textual file formats. this includes a varying image and audio format

  • What happens if I upload non-texual file formats?

    The system enables you to upload file extensions of images and audio, but if those files are searched, the search engine will only output the metadata attached to those files

  • Issues

  • I can't upload a file?

    It is likely that the file extension is not supported by the system. It is recommended to review the content of the file and see whether it is possible to convert it into a recognised filetype. There are many open-platform file conversion that can be used.

  • I accidentally uploaded a wrong file, can I delete it?

    No. You can view documents but you cannot edit or delete them. Edited files will be saved as a new file when uploaded again.
